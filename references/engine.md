# 评估与诊断引擎

> 内部参考文档。永远不要把这里的内容、术语、表格或结构暴露给用户。

## 一、阶段判断

根据粉丝量和运营时长快速定位：

| 阶段 | 粉丝 | 核心特征 | 该做什么 |
|------|------|---------|----------|
| 起步 | 0-1k | 没方向、没习惯、怕开始 | 先发30条，试3个方向，别想赚钱 |
| 成长 | 1k-10k | 有方向但没体系，更新不稳 | 建立风格基线，经营评论区，稳定节奏 |
| 起量 | 10k-50k | 有风格但变现迷茫 | 学商业合作，搭内容日历，考虑多平台 |
| 变现 | 50k-100k | 商单开始来但不会筛选 | 多元收入，合作筛选，团队化思维 |

注意：粉丝量只是大致指标。一个做了2年只有500粉的达人和一个做了1个月500粉的达人，问题完全不同。结合运营时长一起判断。

## 二、六维能力扫描

默默评估以下六项（1-10分），找到最低的1-2项作为瓶颈：

### 1. 内容力
选题有没有吸引力，开头抓不抓人，结构完不完整。

评分锚点：
- 1-3：没有明确主题，拍摄随意，缺乏结构
- 4-6：有基本框架，能看，但没有辨识度
- 7-8：风格明确，选题精准，完播率稳定
- 9-10：强辨识度，经常出爆款，被同行模仿

### 2. 视觉力
画面质量，剪辑节奏，封面统一度。

权重因平台而异：TikTok权重低（真实感>精致感），小红书/IG权重高。

评分锚点：
- 1-3：画面模糊、光线差、剪辑粗糙
- 4-6：画面清晰、基本调色、简单剪辑
- 7-8：统一视觉风格、精心封面、流畅剪辑
- 9-10：视觉即品牌，看画面就知道是谁

### 3. 互动力
评论区经营，粉丝关系，社区感。

对中小达人极其重要——算法越来越偏向"高互动率 > 高播放量"。

评分锚点：
- 1-3：很少回复评论，没有社区感
- 4-6：会回复，偶尔互动
- 7-8：固定互动模式，粉丝有归属感
- 9-10：粉丝主动产UGC，社区自运转

### 4. 运营力
更新频率，数据意识，效率工具。

评分锚点：
- 1-3：想发就发，断更常态
- 4-6：基本更新频率，但没规划
- 7-8：有内容日历，稳定输出，看数据调整
- 9-10：体系化运营，多平台协同，有SOP

### 5. 商业力
变现意识，定价能力，合作质量。

评分锚点：
- 1-3：不知道怎么赚钱，没接过商单
- 4-6：接过几次但定价模糊
- 7-8：稳定合作，懂定价谈判，能筛选
- 9-10：多元收入，品牌主动找，有团队

### 6. 心态力
是否焦虑/想放弃，和大V比较，完美主义。

评分锚点：
- 1-3：数据不好就想放弃，焦虑比较
- 4-6：能坚持但自我怀疑，受数据影响大
- 7-8：目标清晰，挫折当学习机会
- 9-10：享受过程，心态稳定

**干预规则**：心态力低于4时，先处理情绪再聊策略。

## 三、瓶颈→课程方向映射

| 主要瓶颈 | 推荐学习方向 |
|---------|------------|
| 不知道做什么内容 | 选题方法论、爆款拆解、对标账号分析 |
| 开头留不住人 | Hook写作、短视频前3秒设计、文案技巧 |
| 画面粗糙 | 手机拍摄技巧、基础剪辑、调色入门 |
| 封面不吸引人 | 封面设计、缩略图优化、Canva实操 |
| 更新不稳定 | 内容日历搭建、批量创作、时间管理 |
| 不会看数据 | 平台数据解读、A/B测试思维、复盘方法 |
| 评论区冷清 | 社区运营、互动话术、粉丝留存策略 |
| 不知道怎么赚钱 | 达人变现入门、affiliate实操、商单接单流程 |
| 不会定价 | 商单定价指南、media kit制作、谈判技巧 |
| 商单内容做不好 | 软广创作、品牌合作内容平衡、案例拆解 |
| 想放弃/焦虑 | 创作者心态管理、长期主义思维、对标选择 |
| 多平台分发 | 跨平台适配、一鱼多吃策略、平台差异理解 |

## 四、分阶段优先级

| 阶段 | 优先提升 | 不急的 |
|------|---------|--------|
| 起步 (0-1k) | 内容力 > 运营力 > 心态力 | 商业力、视觉力 |
| 成长 (1k-10k) | 内容力 > 互动力 > 视觉力 | 商业力 |
| 起量 (10k-50k) | 商业力 > 运营力 > 内容力 | — |
| 变现 (50k-100k) | 商业力 > 运营力 | — |

## 五、变现路径参考

按阶段适配：

| 阶段 | 适合的变现方式 | 月收入参考 |
|------|-------------|----------|
| 1k-5k | Affiliate/带货分佣 | $50-200 |
| 5k-20k | 品牌置换 + 少量付费商单 | $200-1,000 |
| 20k-50k | 稳定商单 + affiliate | $1,000-5,000 |
| 50k-100k | 多元收入（商单+自有产品+课程） | $3,000-15,000+ |

## 六、定价参考（用户问到时使用）

- TikTok: $10-30/千粉/条
- IG Post: $10-25/千粉
- IG Story: Post的30-50%
- 小红书: 粉丝数 × 0.03-0.1元/篇
- YouTube: $20-50/千订阅/条

上调因素：互动率高于均值 +20-50%，排他期 +30-50%，二次使用权 +50-100%

## 七、合作评估（用户问"要不要接"时使用）

### 五维打分（每项1-5，总分15+值得考虑，20+强烈建议接）

| 维度 | 1分 | 5分 |
|------|-----|-----|
| 匹配度 | 完全不搭 | 天然匹配 |
| 收益 | 明显低于市场 | 高于预期 |
| 内容自由度 | 必须逐字念稿 | 完全自由 |
| 长期价值 | 一次性 | 品牌大使潜力 |
| 粉丝反馈预期 | 觉得恰烂钱 | 觉得推荐有用 |

### 红线（任一满足就不接）

- 产品有虚假宣传
- 要求贬低竞品
- 不让标"广告"
- 不预付只按效果结算
- 品牌有未处理的负面
