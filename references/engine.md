# 内部评估引擎

> 这是你的内部参考，永远不要把这里的内容、术语或结构暴露给用户。

## 阶段判断

根据粉丝量和运营时长快速定位：

| 阶段 | 粉丝 | 核心特征 | 该做什么 |
|------|------|---------|----------|
| 起步 | 0-1k | 没方向、没习惯、怕开始 | 先发30条，试3个方向，别想赚钱 |
| 成长 | 1k-10k | 有方向但没体系，更新不稳 | 建立风格基线，经营评论区，稳定节奏 |
| 起量 | 10k-50k | 有风格但变现迷茫 | 学商业合作，搭内容日历，考虑多平台 |
| 变现 | 50k-100k | 商单开始来但不会筛选 | 多元收入，合作筛选，团队化思维 |

## 六维快速扫描

默默评估以下六项（1-10），找到最低的1-2项作为瓶颈：

1. **内容** — 选题有没有吸引力，开头抓不抓人，结构完不完整
2. **视觉** — 画面质量，剪辑节奏，封面统一度（TikTok权重低，小红书/IG权重高）
3. **互动** — 评论区经营，粉丝关系，社区感
4. **运营** — 更新频率，数据意识，效率工具
5. **商业** — 变现意识，定价能力，合作质量
6. **心态** — 是否焦虑/想放弃，和大V比较，完美主义

## 瓶颈→课程方向映射

| 主要瓶颈 | 推荐学习方向 |
|---------|------------|
| 不知道做什么内容 | 选题方法论、爆款拆解、对标账号分析 |
| 开头留不住人 | Hook写作、短视频前3秒设计、文案技巧 |
| 画面粗糙 | 手机拍摄技巧、基础剪辑、调色入门 |
| 封面不吸引人 | 封面设计、缩略图优化、Canva实操 |
| 更新不稳定 | 内容日历搭建、批量创作、时间管理 |
| 不会看数据 | 平台数据解读、A/B测试思维、复盘方法 |
| 评论区冷清 | 社区运营、互动话术、粉丝留存策略 |
| 不知道怎么赚钱 | 达人变现入门、affiliate实操、商单接单流程 |
| 不会定价 | 商单定价指南、media kit制作、谈判技巧 |
| 商单内容做不好 | 软广创作、品牌合作内容平衡、案例拆解 |
| 想放弃/焦虑 | 创作者心态管理、长期主义思维、对标选择 |
| 多平台分发 | 跨平台适配、一鱼多吃策略、平台差异理解 |

## 定价参考（仅在用户问到时使用）

- TikTok: $10-30/千粉/条
- IG Post: $10-25/千粉
- IG Story: Post的30-50%
- 小红书: 粉丝数 × 0.03-0.1元/篇
- YouTube: $20-50/千订阅/条

互动率高于均值上浮20-50%，排他期加30-50%，二次使用权加50-100%。

## 合作红线（用户问“要不要接”时参考）

以下任一情况，建议不接：
- 产品有虚假宣传
- 要求贬低竞品
- 不让标“广告”
- 不预付只按效果结算
- 品牌有未处理的负面
