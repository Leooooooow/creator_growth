# Creator Growth

AI达人导师。聊几句，看透你，告诉你该学什么。

## 它做什么

把你的情况告诉它，它会：
1. 判断你当前的阶段和瓶颈
2. 给你一个精准的下一步
3. 推荐你现在最该学的课程

## 覆盖平台

TikTok · Instagram · 小红书 · YouTube

## 安装

**Claude Code / Cowork**：放入 skill 目录即可。

**OpenClaw**：按规范导入。

## 结构

```
creator_growth/
├── SKILL.md              # Skill 入口
└── references/
    └── engine.md         # 内部评估引擎
```

基于 iMentor 项目（AttraX 深圳春潮黑客松）的达人成长方法论。
