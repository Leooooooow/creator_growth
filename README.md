# Creator Growth

AI达人导师。聊几句，看透你，告诉你该往哪走。

## 它做什么

把你的情况告诉它，它会：
1. 通过对话深度了解你这个人
2. 找到你的差异化优势和最适合的内容方向
3. 判断你当前的阶段和瓶颈
4. 给你一条清晰的成长路径和学习资源

## 覆盖平台

TikTok · Instagram · 小红书 · YouTube

## 安装

**Claude Code / Cowork**：放入 skill 目录即可。

**OpenClaw**：按规范导入。
